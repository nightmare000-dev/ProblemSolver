# ProblemSolver
This program can solve your math problems. You can use variables `x`, `y` and `z`. You are also able to use `log`, `cos`, `sin`, `tan` and `cot`.
You'll see **Helper page** while running the script

## How to run in Windows
1. Go to the **Releases**.
2. Download file `main.exe`.
3. Run it.

## How to build in Windows
Paste this code into CMD:
```
git clone https://github.com/nightmare000-dev/ProblemSolver
cd ProblemSolver
python -m venv .venv
.\.venv\Scripts\activate.bat
python -m pip install -r requirements.txt
python main.py
```
